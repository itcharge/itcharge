> 是时候构建一套个人账号管理体系了。

[](https://qcdn.itcharge.cn/images/20200820114627.jpg)

## 1. 前言

在这个信息化爆炸的年代里，我们拥有的网络社交账号越来越多。像 QQ、微信、微博、淘宝、京东、知乎、豆瓣、人人、抖音、快手、网易云音乐，甚至 Facebook、Twitter 等等，一大堆账号层出不穷。几乎每个人手上都有几十个甚至上百个账号。

而且这些账号有些是「手机号」注册的，有些是「邮箱」注册的，有些直接是「数字」构成的（例如 QQ），甚至还有以「中文」的方式构成的（例如百度贴吧）。

如果账号注册的时候是临时起意，随便想的一串字符，那很有可能，在下一次登录的时候就忘记了自己的账号。同时，如果在各个平台注册的账号没有规律，也可能在某次登录的时候，会出现多次尝试不同账号的问题。

如果我们可以有效地把各种不同的网络社交账号按照一定的规则进行统一管理，将会解决上述问题，大大提升我们的效率。并且还能做到对外宣传和隐私保护并重，从而在网络世界里更加游刃有余。

* * *

## 2. 账号分类

账号的分类方式有好几种。

 * 依照社交平台的不同，可以将账号分为：QQ、微信、微博、知乎 ......
 * 依照注册类型的不同，可以将账号分为：手机号、邮箱、账户 ID / 用户名、第三方授权（例如微信登录、微博登录等）。
 * 依照隐私需求的不同，可以将账号分为：公开账号、私人账号。



根据不同的分类规则，不同的需求，我们理应选择对应不同的账号。

### 2.1 关于不同平台的注册类型

我们先来说说不同社交平台的注册类型。虽然各个平台的账号类型要求不尽相同，但几乎主流的社交平台都可以使用「手机号」进行账号注册，并且要求实名认证。

另外，还有一些小型网站或者论坛需要通过「邮箱地址」的方式或者使用用户自定义「用户名」的方式进行账号注册。

所以想要进行完整的账号管理，我们需要准备好「手机号」、「邮箱」和自己定义的「账号 ID / 用户名」，以完成各个平台的账号注册。

### 2.2 关于隐私问题

我们再来说一下关于账号的隐私问题。对于我们大多数人来说，根据隐私需求的不同，可以将账号分为：**「公开账号」** 、**「私人账号」** 。

#### 2.2.1 公开账号

「公开账号」就是你想要展示给大众的账号。现在流行一个词，叫做「个人品牌」。意思是把自己当作一个产品，一个品牌来经营。向外输出自己的想法、价值观思维方式，从而提升自身的影响力，实现自己的个人价值，并有可能创造可观的商业利润。

你的公开账号，在某种程度上，也算是你在互联网上的个人品牌和数字名片，比如现在网络上很火的「李子柒」、「朱一旦」、「李佳琦」、「papi 酱」。如果你是一个内容输出者，或者活跃的社交达人，个人建议，不妨花一定的时间，好好设计规划一下长期的「公开账号」，取一个好听的名字，一个有意义的、通俗易懂的、辨识度高、便于大家熟记的账号名称。并且尽量保持各个平台上的账号统一，形成自己的独特标识。

比如说我自己的「公开账号」用户名就是「程序员充电站」。通过搜索引擎，可以很容易搜索到我的「个人博客」、「简书」、「GitHub」、「CSDN」、「知乎」等各个平台的链接。

[](https://qcdn.itcharge.cn/images/20220802100444.png)

#### 2.2.2 私人账号

「私人账号」跟公开账号相反，是你不想被别人看到，或者说不愿意展示给大众的账号。

###### 1.「私人账号」的第一个作用是活出真实的自我。

每个人都是多面的，而展现给大众所看到的，只是其中的几个方面。为了维持一些特定的形象，在微信朋友圈、微博里，我们也不得不放弃一些喜好和性格。但是，人类都是有表达和倾诉需求的，这时候「私人账号」就起到了作用。比如说想要发条负能量的微博，又不好在「公开账号」里发表，这时候申请的微博小号就有了用处。或者说想吐槽一下工作中的不顺，又不想老板知道，这时候申请的微信私人账号就有了用处。私人账号有时候是另一种生活方式，可以让我们在自己的世界里卸下伪装。

###### 2. 「私人账号」另一个作用是免受信息打扰。

**万物互联的时代，个人隐私成为奢侈。**

当你发现用自己手机号注册过某 App 之后，在之后一段时间里，自己不断收到各种推销骚扰电话，你就会发现拥有一个「手机小号」的重要性。强烈建议使用一个专门的手机小号。大号用于亲友联络，小号则用来来管理各类 App 注册信息。这样能减少很多无用信息的打扰。

## 3. 不同注册类型账号管理

上面一节中我们说了，想要进行完整的账号管理，我们需要准备好「手机号」、「邮箱」和自己定义的「用户名」，以完成各个平台的账号注册。下面，我们来分别讲解一下。

### 3.1 账号 ID / 用户名

账号 ID / 用户名建议按照隐私需求的不同，准备至少两个：一个「公开账号」，一个「私密账号」。

 * 「公开用户名」要求：好听，有意义，通俗易懂，辨识度高、便于大家熟记辨识度高；尽量保持各个平台上用户名统一。
 * 「私人用户名」要求：尽量与公开用户名不同；尽量使各个平台的用户名有所区分，比如说 zhangsan001，zhangsan002。



### 3.2 手机号

因为国内「实名制」的强制政策，还有「手机号」注册的便利性条件，现在国内几乎所有的社交平台都需要使用手机号进行注册或绑定。所以我们需要至少一个手机号码。

而基于上一节我们说的隐私问题，这里我建议至少准备两个手机号来应对不同的需求。有条件的甚至可以再准备一个境外手机号，用于境外服务注册。

 * 主号码（私人）：仅用于与亲友联系，以及支付宝，银行等相关重要账号的绑定。
 * 副号码（公开）：用于各种 App、网站注册，快递外卖联络等。
 * 境外号码：用于境外服务注册，非实名注册用。



> 关于主、副号码的套餐问题，可以用主号码选择流量高的套餐作为日常使用，副号码选择低月租的保号套餐。

### 3.3 邮箱号

邮箱号同样也用于某些网站注册和绑定，这里同样建议至少准备两个以上的邮箱号。有能力者建议准备一个境外邮箱（推荐 Gmail）。

 * 主邮箱（私人）：仅用于熟人联系，以及支付宝、学信网等重要账号的绑定。
 * 副邮箱 1（公开）：仅用于对外沟通。
 * 副邮箱 2（公开）：用于各种 App、网站注册。
 * 境外邮箱：用于境外服务注册，非实名注册用。



#### 3.3.1 邮箱服务商选择

由于涉及到账号的注册和绑定，一个邮箱往往要使用很久。选择一个稳定、安全的邮箱服务是十分必要的。目前免费邮箱众多，可选择的有：QQ 邮箱、163 邮箱、sina 邮箱、Outlook 邮箱、Gmail 邮箱。

国内推荐 QQ 邮箱 / 163 邮箱，境外推荐 Gmail 邮箱。

#### 3.3.2 邮箱号 ID 选择

我们在注册邮箱号 ID 的时候，一定要选择合适的 ID。

###### 1. 切勿将 QQ 号 / 手机号设置为邮箱号 ID。

诸如 `2362XXX63@qq.com` 或者 `138002XXX63@139.com` 之类很容易暴露隐私安全（QQ 号、手机号等）。别人很容易通过邮箱号来获取你的其他个人信息，从而导致信息泄露。

###### 2. 选择合适的邮箱 ID。

 1. 注册用于熟人联系的主邮箱 ID 时，建议优先使用个人姓名的拼音，或者英文名。
 2. 注册用于对外沟通的副邮箱 ID 时，建议与上边提到的公开账号 ID / 用户名相同。
 3. 注册用于各种 App、网站注册的账号时，建议与主邮箱 ID 区分开。
 4. 更极致的做法是申请域名邮箱，具体操作见百度。



#### 3.3.3 多个邮箱号管理

虽然上边提到的邮箱服务都可以通过浏览器打开自己的邮箱。但是还是建议使用专门的邮件管理客户端来管理邮件。这样的好处是可以随时随地的接受到邮件。无论你是在公司的电脑上，还是在家里的电脑上，还是在自己的手机上，都可以实时的接受到邮件信息。

关于邮件管理客户端，推荐使用「网易邮箱大师」。iOS、Android、 Windows、Mac 上都有对应软件。

## 4. 域名（个人网站）

最后来说一下 **「域名」** 。

「域名」是一台或者一组服务器的名称，它是由一串用点分隔的名字组成的，一般有前缀和后缀。「域名前缀」一般是在注册时由注册者自定义的；「域名后缀」则是在注册的时候由注册者选择的。像 `.com`、`.net`、`.org` 这种都是「域名后缀」，也被称为「顶级域名」。我们可以将「域名」简单理解为网站（并不标准）。比如 QQ 网站 `www.qq.com` 的域名就是 `qq.com`，域名前缀是 `qq`，域名后缀是 `com`。

即便是现在有各种 App、微博、微信公众号、小程序，但是网站依然是互联网内容中不可或缺的载体。因此，如果你想要尝试通过互联网来宣传和推广自己，可以考虑购买一个「域名」，并且尝试一下搭建自己的网站。

因为个人网站一般是用来对外展示自己，所以我们需要一个公开的域名。建议注册域名时，前缀可与上边提到的公开账号 ID / 用户名相同。

而后缀的选择这里提供一些我的个人建议：

 * 优先选择 `.com` / `.net` 等顶级域名。如果你想要的域名 `.com` / `.net` 都没有了，那选择 `.org` 或者 `.cn` 也可以。比如我的网站 `itcharge.cn`用的就是 `.cn` 域名。
 * 上述域名没有了的话，选择`.top`、`.link`、`.site`、`.name`、`.cc`、`.info` 等常见域名。或者一些小国家域名，例如`.me`、`.co`、`.io`、`.im` 等也可以。
 * 域名尽量简短有特点，好记，且辨识度高，这和账号 ID / 用户名要求是差不多的。



* * *

## 5. 总结

现在我们完成了个人账号体系的管理和建设，将多个平台的账号逐一打通。既使用统一的公开账号对个人品牌进行了宣传，又使用相对私密的私人账号对个人隐私做了保护。

构建一套个人账号管理体系，对自己的众多账号统一整理和规划，实际上也是对生活的一种整理和规划。将外在展示内容和个人生活内容做分割处理，才能在宣传自己的个人品牌的同时，提升个人生活的品质。

**毕竟，生活中更好的经验都是为了创造更美好的生活。**
